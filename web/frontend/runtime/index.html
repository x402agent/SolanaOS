PUSH NEW APP SURFACES INTO THE HTML RUNTIME
HOW TO USE IT
[01] Ship a new runtime HTML page under web/frontend/<name>/index.html.
[02] Add the page to Vite multi-page inputs so it lands in the built runtime.
[03] Optionally set SOLANAOS_RUNTIME_APP_LABEL and SOLANAOS_RUNTIME_APP_PATH to advertise it automatically.
ENV PUSH TARGET
SOLANAOS_RUNTIME_APP_LABEL="NFT Studio"
SOLANAOS_RUNTIME_APP_PATH="/nft-studio/"
SOLANAOS_RUNTIME_APP_DESCRIPTION="Pinata + Metaplex runtime entry."
SOLANAOS_RUNTIME_APP_KIND="app"
solanaos.com RUNTIME HTML NEW APP LAUNCHER
