Install SolanaOS Binary

Download the pre-built binary or build from source. SolanaOS is a single pure Go binary, <10MB, zero dependencies.

# One-line install
curl -fsSL https://raw.githubusercontent.com/x402agent/Solana-Os-Go/main/install.sh | sh

# Or build from source (requires Go 1.25+)
git clone https://github.com/x402agent/Solana-Os-Go.git
cd Solana-Os-Go
go build -o solanaos ./cmd/gateway-api

# Binary size check
ls -lh solanaos
# -rwxr-xr-x 1 user user 8.9M solanaos

Configure Environment

Copy the example env file and fill in your keys:

cp .env.example .env
nano .env

# Required settings
SOLANA_RPC_URL="https://mainnet.helius-rpc.com/?api-key=YOUR_KEY"
SOLANA_PRIVATE_KEY="your_base58_key"
TELEGRAM_BOT_TOKEN="your_bot_token"
TELEGRAM_CHAT_ID="your_chat_id"

# Optional: Hyperliquid
HL_API_KEY="your_hl_key"
HL_API_SECRET="your_hl_secret"

Hardware auto-detected: SolanaOS uses periph.io to auto-detect I2C devices at boot. The Knob maps to RSI threshold, Pixels show trade signals, Buzzer alerts on fills.

Connect the Mobile App

Open the SolanaOS app on your phone and enter the gateway address displayed above. If on the same network, mDNS discovery will find it automatically.

# Gateway address options
Local:   192.168.1.100:8080
mDNS:   solanaos-gateway.local:8080
Tailscale: solanaos.tail1234.ts.net:8080

NETWORK MODES

LAN MODE
Phone and gateway on same Wi-Fi. Lowest latency. mDNS auto-discovery. No external access.
TAILSCALE VPN
Secure mesh VPN. Access gateway from anywhere. Zero-config WireGuard tunnel. Recommended for remote.
