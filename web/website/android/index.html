Solana Mobile Stack: The app uses the Solana Mobile Stack (SMS) for wallet integration. Seeker devices have native SMS support. Standard Android devices use the SMS adapter via Phantom/Solflare.

BUILD FROM SOURCE

Clone the Repository

git clone https://github.com/x402agent/Solana-Os-Go.git
cd Solana-Os-Go/apps/android

Configure Local Properties

Create or update local.properties with your Android SDK path:

# local.properties
sdk.dir=/Users/YOUR_USER/Library/Android/sdk

# Or on Linux
sdk.dir=/home/YOUR_USER/Android/Sdk

Build the APK

TERMINAL
> ./gradlew assembleDebug
BUILD SUCCESSFUL in 2m 34s
47 actionable tasks: 47 executed

APK: app/build/outputs/apk/debug/app-debug.apk

For a release build:

# Release build (requires signing config)
./gradlew assembleRelease

# Or use the Makefile
make build

Install on Device

Enable USB debugging on your Android device, then install via ADB:

# Install debug APK
adb install app/build/outputs/apk/debug/app-debug.apk

# Or download the latest release from GitHub
open https://github.com/x402agent/Solana-Os-Go/releases/latest
Seeker Devices: If you have a Saga or Seeker, the app automatically detects SMS and uses the hardware secure element for transaction signing.

Configure the App

On first launch, the app will prompt you to:

[1] Connect a Solana wallet (Phantom/Solflare/Backpack)
[2] Enter your gateway address or scan QR code
[3] Fund the agentic wallet with SOL
[4] Enable OODA autonomous trading
INCLUDED: SOLANA PAY SAMPLE

The solana-pay-android-sample-main/ directory contains a reference implementation for Solana Pay on Android. This enables QR-code based SOL and SPL token transfers directly from the app.

APP FEATURES

DASHBOARD
Real-time wallet balance, OODA cycle status, companion state, trading performance charts.
AGENT TERMINAL
Full terminal access: /status, /trending, /hl_positions, /swap, and all SolanaOS commands.
OODA RUNTIME
Observe, Orient, Decide, Act loop. Strategy config, RSI/EMA/ATR parameters, stop-loss.
HARDWARE
Modulino I2C: Pixels, Buzzer, Knob, IMU, ToF sensor. Real-time hardware monitoring.
