SETUP STEPS

Install the APK

Download the SeekerClaw APK from GitHub releases or build from source. The APK is Solana Mobile Stack compatible and works on any Android 10+ device, with native Seeker support.

# Open the latest GitHub release page
open https://github.com/x402agent/Solana-Os-Go/releases/latest

# Or build locally and install via ADB
adb install apps/android/app/build/outputs/apk/debug/app-debug.apk

Connect Your Wallet

Open the app and tap { CONNECT & PAIR }. Select your preferred wallet (Phantom, Solflare, or Backpack). The app will request transaction signing permissions for the agentic wallet.

Agentic Wallet: SolanaOS auto-generates a local keypair for autonomous trading. This is separate from your main wallet. Fund it with a small amount of SOL to enable OODA trading.

Configure Gateway Connection

Enter your gateway's local IP or Tailscale hostname. The app discovers SolanaOS runtimes on your network automatically via mDNS.

# Default gateway address
Gateway: 192.168.1.100:8080

# Or via Tailscale
Gateway: solanaos-gateway.tail1234.ts.net

Pair & Verify

The app will handshake with your gateway and display the dashboard. You should see your agent wallet address, SOL balance, OODA cycle status, and companion state.

Enable OODA Trading

Once connected, enable the autonomous OODA trading loop from the dashboard. Configure your strategy parameters: RSI thresholds, EMA periods, stop-loss, and position sizing.
