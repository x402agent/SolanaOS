TELEGRAM BOT

REMOTE COMMAND & CONTROL VIA MAWDBOT
MAWDBOT
The Telegram bot gives you full remote access to your SolanaOS runtime. Check balances, monitor trades, view trending tokens, manage Hyperliquid positions, and interact with your companion — all from Telegram.

AVAILABLE COMMANDS

MAWDBOT COMMANDS
/status — Runtime status, wallet, balance, uptime
/trending — Top trending tokens on Solana
/hl_positions — Hyperliquid open positions & PnL
/hl_balance — Hyperliquid account balance
/swap <token> <amount> — Jupiter swap via agentic wallet
/price <token> — Current price of any Solana token
/portfolio — Full token portfolio overview
/companion — Companion state, mood, XP, evolution
/ooda — OODA loop status & current cycle
/memory — Honcho memory context
/hardware — Connected I2C hardware status
/help — List all available commands

SETUP GUIDE

Create a Bot with BotFather

Open Telegram and message @BotFather. Create a new bot:

TELEGRAM
You: /newbot
BotFather: Choose a name for your bot.
You: SolanaOS MawdBot
BotFather: Choose a username for your bot.
You: solanaos_mawdbot
BotFather: Done! Your token is:
7234567890:AAHxxxxxxxxxxxxxxxxxxxxxxxxx
Keep your token secret! Anyone with the token can control your bot. Never commit it to git or share publicly.

Get Your Chat ID

Message your new bot, then use the Telegram API to find your chat ID. This restricts the bot to only respond to you.

# Get your chat ID (after messaging the bot)
curl "https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates" | jq '.result[0].message.chat.id'

Configure Environment

Set the bot token and chat ID in your SolanaOS environment:

# .env file
TELEGRAM_BOT_TOKEN="7234567890:AAHxxxxxxxxxxxxxxxxxxxxxxxxx"
TELEGRAM_CHAT_ID="123456789"

# Optional: set allowed user IDs (comma-separated)
TELEGRAM_ALLOWED_USERS="123456789,987654321"

Start the Bot

The Telegram bot starts automatically when SolanaOS boots. Verify it's running:

SOLANAOS
> ./nanosolana start
✓ Runtime started
✓ Wallet loaded: AAqk...DwaM
✓ Telegram bot connected
✓ OODA loop active: cycle #247
✓ Honcho memory initialized
✓ Hyperliquid bridge ready
