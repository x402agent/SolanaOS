## Copy Direction

- Product name in art: `SolanaOS`
- Subtitle tone: `local-first`, `Seeker-native`, `Solana control surface`
- Avoid `OpenClaw` references in any exported store art

## Submission Usage

Use these exact files in the Solana Mobile dApp Store form:

1. `icon-512.png` for the dApp icon
2. `banner-1200x600.png` for the detail-page banner
3. `preview-01.png` through `preview-04.png` for the required preview gallery

If the artwork changes later, replace the files in this directory and update `docs/SOLANA_MOBILE_SUBMISSION.md` in the same change.
