# Weather

Use this skill for straightforward weather checks.

## Commands

```bash
curl -s "wttr.in/New+York?format=3"
curl -s "wttr.in/San+Francisco?format=%l:+%c+%t+%w"
curl -s "wttr.in/London?format=v2"
```

Always include a city, region, or airport code in the query.
