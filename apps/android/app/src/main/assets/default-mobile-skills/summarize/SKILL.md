# Summarize

Use this skill for quick summaries and transcript extraction.

## Quick start

```bash
summarize "https://example.com"
summarize "/path/to/file.pdf"
summarize "https://youtu.be/example" --youtube auto
```

Prefer short summaries first on mobile, then expand only the sections the user asks for.
