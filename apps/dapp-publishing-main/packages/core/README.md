# @solana-mobile/dapp-store-publishing-tools

Core publishing utilities used by `@solana-mobile/dapp-store-cli`.

This package contains the validation, creation, and publishing helpers used to:

- create app metadata
- create release metadata
- validate local publishing state
- submit update, remove, and support actions to the publisher portal
