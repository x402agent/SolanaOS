# Agent Instructions

Keep repository instructions generic and safe for open source.

This repo is a frontend for SolanaOS. Keep any SolanaOS runtime checkout separate from this repository.

Do not modify the SolanaOS source code. When the user asks for changes, they are asking for changes to this app. Your solutions should be applied to this app but to understand the full context of implementing your solution, you will need to search through SolanaOS's source code.

If you use local private overlay instructions, keep them outside the repository and do not commit them here.

Do not commit personal, environment-specific, or secret instructions to this repository.
