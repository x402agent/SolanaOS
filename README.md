## Features

### Desktop Environment
- **Draggable Windows**: Multi-window support with minimize, maximize, and close controls
- **Start Menu**: Windows 95-style menu with organized application categories
- **Taskbar**: Real-time application management and system status
- **Desktop Icons**: Quick access to frequently used applications
- **Animated Background**: Dynamic particle effects for visual appeal

### SPARK Intelligence Layer
The self-improving AI loop that powers everything:

| Phase | Function |
|-------|----------|
| **OBSERVE** | Monitor user actions, chain state, market data |
| **DECIDE** | Claude-powered reasoning about next best action |
| **ACT** | Execute on-chain transactions, open windows, prefetch data |
| **MEASURE** | Track action outcomes and user satisfaction |
| **LEARN** | Pattern recognition across sessions |
| **IMPROVE** | Self-evolving prompts and workflows |

### Built-in Applications

| App | Description |
|-----|-------------|
| **COMET Chat** | AI-powered blockchain assistant with intent understanding |
| **DEX Panel** | Token swapping via Jupiter aggregator with real-time quotes |
| **Agent Studio** | Create and deploy autonomous trading agents |
| **Portfolio** | Track holdings, performance, and allocation |
| **Chain Explorer** | Watch addresses, decode transactions, trace token flows |
| **Terminal** | Command-line interface with skills installation |
| **SPL Burner** | Permanently remove unwanted tokens and NFTs |
| **Paint Studio** | Digital art creation with IPFS and NFT minting |

## Quick Start

### Prerequisites
- Node.js 18+
- pnpm (recommended) or npm
- A Solana wallet (Phantom, Solflare, Backpack)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/SolanaOS.git
cd SolanaOS

# Install dependencies for web frontend
cd solana-os-web
npm install

# Start development server
npm run dev
```

### Environment Variables

Create a `.env` file in the project root:

```env
# API Keys
VITE_HELIUS_API_KEY=your_helius_api_key
VITE_OPENAI_API_KEY=your_openai_api_key
VITE_BIRDEYE_API_KEY=your_birdeye_api_key

# Network Configuration
VITE_SOLANA_NETWORK=mainnet-beta
VITE_RPC_URL=https://api.mainnet-beta.solana.com
```
