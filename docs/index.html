PUBLISH YOUR SKILL TO NANOHUB

nanohub publish
# 1) Login to NanoHub
npx @solanaos/nanohub login
# 2) Publish a local skill folder (must include SKILL.md)
npx @solanaos/nanohub publish ./my-skill --slug my-skill --name "My Skill" --version 1.0.0 --tags latest,solana

Requirements: SKILL.md (or skills.md), text files only, semver version. Published skills appear on seeker.solanaos.net.

WHAT HAPPENS

1

Clone & Build

Downloads the SolanaOS repo and builds the solanaos 10MB binary in Go.

2

Agentic Wallet

Auto-generates a Solana keypair at ~/.nanosolana/wallet/ — your agent's on-chain identity.

3

Connect APIs

Add your HELIUS_API_KEY and BIRDEYE_API_KEY to the .env file for live mainnet data.

4

Start Trading

Run solanaos daemon for the full autonomous OODA loop, or solanaos ooda --sim for paper trading.
